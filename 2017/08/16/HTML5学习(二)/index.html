地理位置

  1. 第一个参数，一个函数接受一个getCurrentPosition()传过来的一个对象，里面储存着获取到的地理位置

web存储

三种存储方式

  1. cookie()
  2. sessionStorage
  3. localStorage

cookie

  • 只能存储4k，
  • 每次访问同源网站的时候都会在请求和响应中带上这个包袱，一般只存储加了密的用户信息

  • 以字符串的形式存在

    /jq方法/
    //设置cookie
    $.cookie(“key”,”value”)
    //获取cookie
    $.coolie(“key”)
    //删除cookie
    $.removeCookie(“name”)

sessionStorage

window.sessionStorage

  1. 声明周期为关闭浏览器窗口
  2. 不能再多个窗口共享数据
  3. 大小为5m

localStorage

window.localStorage

  1. 永久生效
  2. 可以多个窗口共享
  3. 大小为20m

sessionStorage 和 localStorage 的方法

setItem(key,value)
getItem(key)
removeItem(key)
clear()
key(n)以索引来获取存储内容，获取到的是key，不是value

文件读取

通过FileReader对象我们可以读取本地存储的文件，可以使用 File 对象来指定所要读取的文件或数据。其中File对象可以是来自用户在一个
元素上选择文件后返回的FileList 对象，也可以来自由拖放操作生成的 DataTransfer

files 对象

对于file类型的input框，在dom中存在files属性，这个属性保存的是一个叫filelist的对象，是一个伪数组，里面保存这上传的文件指定multiple可以上传多个
