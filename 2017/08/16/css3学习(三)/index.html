css3 学习(三 弹性盒子)

发表于 2017-08-16

弹性盒子

声明一个弹性盒子*

display:flex;

只要给一个盒子设置来人display:flex,他就有了主轴和侧轴子元素是默认在主轴上从左到右排列

给父盒子设置的属性

flex-direction调整主轴的方向

默认是row 行

  1. row 默认从左到右
  2. row-reverse 从右到左
  3. clumn 从上到下
  4. clumn-reverse 从下往上

justify-content 调整子元素在主轴方向的对齐方式

  1. flex-start 主轴方向排
  2. flex-end 弹性盒子元素将向行结束位置对齐。该行的第一个子元素的主结束位置的边界将与该行的主结束位置的边界对齐，同时所有后续的伸缩盒项目与其前一个项目对齐。
  3. center 居中对齐
  4. space-between 第一个帖左边，最后一个贴右边，其他的均分，常用
  5. space-around 完全均分(盒子左右有空隙)

align-items 侧轴排列(不换行时候的侧轴对齐方式)

  1. flex-start
  2. flex-end
  3. center
  4. stretch 拉伸
  5. beseline

align-content 侧轴对齐方式（换行时）

  1. flex-start
  2. flex-end
  3. center
  4. stretch 拉伸
  5. beseline
  6. space-between 第一个帖左边，最后一个贴右边，其他的均分，
  7. space-around

flex-wrap

  1. nowrap： flex容器为单行。该情况下flex子项可能会溢出容器
  2. wrap：flex容器为多行。该情况下flex子项溢出的部分会被放置到新行，子项内部会发生断行

给子元素设置的属性

flex 如何分配主轴空间

order 设置子元素的排列循序，值越小，越靠前

align-self 定义flex子项单独在侧轴（纵轴）方向上的对齐方式。
